# auth-redux

## Run Server

To run the server, follow these steps:

1. Navigate to the `backend` directory:

   cd backend
   npm run start
   
## Run Frontend 

To run the frontend, follow these steps:

2. Navigate to the `frontend` directory:

   cd frontend
   npm run dev
   
   
## Run The Database
3. Create a MongoDB database with the name auth-redux
